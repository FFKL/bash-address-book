# Address Book

Address book written in bash

```
      ______ ______
    _/      Y      \_
   // ~~ ~~ | ~~ ~  \
  // ~ ~ ~~ | ~~~ ~~ \      Welcome to
 //________.|.________\     The Address Book
'----------'-'----------'
 (1) Search
 (2) Add
 (3) Edit
 (4) Remove
 (l) Show full list
 (q) Quit
```
## Running
``` bash
chmod u+x ./app/addr.sh # make main script executable
./app/addr.sh
```
